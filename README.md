# Chalkoholics — Wednesday 8-Ball Hub

Live stats hub for the Society Billiards Wednesday 8-ball league (team **Chalkoholics**).

**View it:** https://nlplofker.github.io/chalkoholics-hub/

`index.html` is a single self-contained page (all CSS, JS, and data inlined — no build, no backend).
It's regenerated from the source in `~/ounce/pool-hub/` and refreshed as new match results come in
(see that project's `README.md` / the `pool-hub-update` Claude skill). To update: replace `index.html`
and push — GitHub Pages redeploys automatically.
